The Three Main Image Formats On The Web...

PNG-

Portable Network Graphics (a.k.a PNG) is a compressed raster graphic format.

GIF-

Graphics interchange Format(a.k.a GIF) is an image file format commonly used on the web and sprites for software programs.

JPEG

Joint Photographic Experts Group (a.k.a JPEG) is a popular file format.
